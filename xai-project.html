Explainable AI visualizations, model interpretations, and transparency techniques.

This project focuses on traffic violation detection using Artificial Intelligence and computer vision. The model is trained to identify real-time violations such as helmetless riding and triple riding using object detection techniques. To make the system transparent and explainable, Grad-CAM visualizations are used to highlight the exact regions of the image the model focuses on while making predictions. This helps ensure credibility and explainability, especially in safety-critical applications like smart traffic monitoring and law enforcement automation.

The system is built using YOLOv8, TensorFlow/Keras, OpenCV, and Python, and can be further extended for real-time CCTV surveillance or integration into Smart City platforms.
